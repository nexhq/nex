Getting Started with nex

Your journey to better tool management starts here.

Installation

Install nex on your system using one of the following methods:

Windows

Download the installer or run via PowerShell:

iwr https://github.com/nexhq/nex/releases/latest/download/install.ps1 -useb | iex

Linux / macOS

Run the installation script:

curl -fsSL https://github.com/nexhq/nex/releases/latest/download/install.sh | sh

First Steps

Once installed, verify the installation by checking the version:

nex --version

Installing a Package

Try installing a package, for example, the hello-world package:

nex install hello-world

Quick Examples

Here are some common commands to get you moving:

  • nex search <query> - Search for packages
  • nex list - List installed packages
  • nex update <package> - Update a package
