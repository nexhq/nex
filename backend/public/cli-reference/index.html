CLI Reference

Comprehensive documentation for all nex commands.

install

nex install <package> [version]

Installs a package from the registry. If no version is specified, the latest version is installed.

remove

nex remove <package>

Removes an installed package from your system.

update

nex update <package>

Updates an installed package to the latest version.

list

nex list

Lists all installed packages and their versions.

search

nex search <query>

Searches the registry for packages matching the query.

info

nex info <package>

Displays detailed information about a package.

run

nex run <package> [args...]

Runs a package command without installing it globally (if supported).
